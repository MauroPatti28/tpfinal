### Sistema de Gestión para Restaurante "El Parador"

**Descripción:**  
Desarrollo completo de una aplicación web para el restaurante "El Parador" en San Miguel de Tucumán. La solución incluye:

1. **Panel Administrativo:**
   - CRUD completo para platos (con imágenes, etiquetas dietéticas y precios)
   - Gestión de reseñas de clientes
   - Dashboard intuitivo con operaciones en tiempo real

2. **Frontend para Clientes:**
   - Página "Acerca de Nosotros" con diseño visual impactante
   - Sistema de reseñas interactivo
   - Menú digital con filtros (apto celíacos, veganos, etc.)
   - Chatbot integrado para atención al cliente

3. **Funcionalidades Clave:**
   - Autenticación segura JWT para administradores
   - Integración con WhatsApp para reservas
   - Animaciones con React Intersection Observer
   - Diseño responsive con Material-UI y CSS personalizado
   - Sistema de etiquetas dietéticas preconfiguradas

**Tecnologías Principales:**
- React 18
- Node.js/Express
- MongoDB
- Material-UI
- React Dropzone (gestión de imágenes)
- JWT Authentication
- Axios para comunicación API

**Características Destacadas:**
- Animaciones scroll-trigger en secciones "Acerca de"
- Panel admin con carga/edición de imágenes en tiempo real
- Sistema de reviews con moderación
- Chatbot con historial conversacional persistente
- WhatsApp Button con tooltip interactivo

**Enfoque UX:**
- Navegación intuitiva con categorización de platos
- Feedback visual en operaciones CRUD
- Transiciones suaves entre componentes
- Diseño accesible con contraste adecuado

---

**Nota para portfolio:**  
Este proyecto demuestra habilidades full-stack con énfasis en UX, gestión de estado complejo (CRUD con imágenes) e integración de APIs. Resalta la solución de problemas reales como la gestión de menús dinámicos y comunicación omnicanal con clientes.
